What it does

Built to make public health data
actually public.

🗺️

Interactive Map Dashboard

A visual, state-by-state choropleth map showing COVID case counts and vaccination progress across the US — color-coded by coverage for fast pattern recognition.

💉

Live Vaccine Statistics

Real-time data on total doses administered, percentage of population vaccinated, and breakdown by dose number — pulled directly from the CDC's public data API.

🏭

Brand-Specific Profiles

Detailed fact cards for each authorized vaccine — Pfizer-BioNTech, Moderna, and J&J — covering efficacy rates, dosing schedules, and distribution share.

📊

Case Trend Comparison

Side-by-side view of case counts versus vaccination rates, revealing the relationship between rollout progress and outbreak trajectories over time.

How it works

From raw API data
to insight in four steps.

01
Fetch from CDC API

On launch, the app makes HTTP requests to CDC public endpoints to pull the latest vaccination and case records for every US state and territory.

02
Parse & structure the data

The JSON response is mapped into structured Java objects — one per state — containing case totals, dose counts, and brand-level distribution figures.

03
Render the map

Data is fed into the map layer, rendering color-coded state regions. Click and hover interactions surface detailed per-state statistics.

04
Explore brand details

A separate panel surfaces brand-specific fact sheets — combining live distribution data with curated vaccine information for full context.

What it looked like
when it ran.

Step 1
Open the map

The app launches on the Map View tab with all 50 states color-coded by vaccination rate.

Step 2
Click a state

Hover or click any state to see case count, vax rate, and daily dose totals in the sidebar.

Step 3
Switch to Vaccine Info

The Vaccine Info tab shows brand-specific fact cards for Pfizer, Moderna, and J&J.

Step 4
View trends

The Trends tab charts case counts against vaccination rates over time to show rollout impact.
